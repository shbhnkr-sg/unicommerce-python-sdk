# Changelog

## [0.1.0] - Unreleased

### Added
- Initial release of the Unicommerce Python SDK
- OAuth 2.0 authentication with auto token refresh
- Sync and async client support
- Resource-based API: sale_orders, inventory, products, fulfillment, inbound, returns, facilities, export_jobs
- Custom exception hierarchy
- Automatic retry with exponential backoff for safe operations
- Pydantic models for all request/response types
